# ZenPod — Mindful Work App Prototype
**Scenario 1: Build Mindful Work App Prototype for Stress-Free Offices**

## Project Overview
**Project Title:** ZenPod — Smart Stress Relief Pod App
**Target Audience:** Office employees across all roles and age groups
**Scenario:** FutureForward Wellness wants to help employees manage workplace stress through a Smart Stress Relief Pod, managed via a mobile app prototype.
**Tools Used:** Figma (prototyping), Miro (empathy maps + ideation), Canva (persona visuals), MindMeister (mind mapping)

---

## Design Thinking Process

### Stage 1: Empathize & Define
- Conducted research with 10+ participants (surveys, interviews, observations)
- Identified key stress triggers: tight deadlines, back-to-back meetings, lack of private calm spaces
- Created 10 detailed user personas covering diverse ages (22–60), job roles, stress levels, and tech comfort
- Built empathy maps for 3 key personas (Riya, Aarav, Kavita)
- **Problem Statement:** "How might we design a smart, personalized, and instantly accessible stress-relief experience for employees that fits within a short work break?"
- 📁 See: /Docs/personas.txt | /Docs/empathy-maps.txt | /Docs/research-summary.txt

### Stage 2: Ideate
- Brainstormed features using Miro mind maps and "How Might We" questions
- Prioritized 7 features including all 5 compulsory ones
- **5 Compulsory Features:** MoodSync AI | Breathe With Me | Pod Scheduler | Post-Session Journal | Stress Tracking Dashboard
- **Additional Features:** Wellness Reward Badges | Emergency Calm Button
- 📁 See: /Docs/ideation.txt

### Stage 3: Prototype
- Created 7 clickable screens in Figma
- Covers: Login/Dashboard, MoodSync AI Check-in, Pod Control, Guided Breathing, Scheduler, Journal, Dashboard
- 🔗 [Prototype Link — https://www.figma.com/make/Oo87sFKVChbzGWNcdzljnS/Mindful-Work-App-Prototype?t=e676AhRDPKodT5AV-1
- 📁 See: /Wireframes/

### Stage 4: Test
- Tested with 8 users across different age groups and roles
- Average satisfaction: 4.3 / 5
- Key improvements: larger slider text, clearer navigation icons, emoji-based stress scale, first-time user tour
- 📁 See: /Docs/testing-report.txt
